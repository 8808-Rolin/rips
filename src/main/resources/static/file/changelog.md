## 项目概述

- 基于 SpringBoot 实现的跨平台Web自助打印平台。
- 核心模块包括：打印、扫描、排版、生成内容等多方面内容
- 代码量少、学习简单、功能强大、易扩展，轻松开发从现在开始！
